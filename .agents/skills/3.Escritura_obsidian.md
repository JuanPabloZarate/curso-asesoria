# Markdown de Obsidian

Obsidian utiliza una potente extensión de Markdown diseñada para el pensamiento en red.

## Sintaxis Principal

- **Wikilinks**: `[[Nombre de la Nota]]` para enlaces internos.
- **Incrustaciones**: `![[Nombre de la Nota]]` para transcluir contenido.
- **Callouts**: `> [!INFO]` para cuadros con estilo.
- **IDs de Bloque**: `^id-de-bloque` para enlaces granulares.
- **Notas al pie**: `[^1]` para citas.

## Metadatos (YAML)

Incluye siempre un bloque de frontmatter en la parte superior del archivo:
```yaml
---
tags: [estado/activo, tema/obsidian]
created: 2024-03-20
---
```

## Mejores Prácticas

1. **Notas Atómicas**: Mantén las notas centradas en un solo concepto.
2. **Enlaces Densos**: Usa `[[ ]]` para crear una red de conocimiento.
3. **Etiquetas Consistentes**: Usa etiquetas para el estado (ej. `#todo`) o el tipo (ej. `#persona`).

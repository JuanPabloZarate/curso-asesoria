# Conversor de Archivos a MD

Este skill te permite extraer información estructurada a partir de cualquier archivo o texto en crudo (por ejemplo, correos electrónicos, documentos PDF, notas de reuniones u otros temas) y convertirlo en un archivo Markdown (`.md`) para la base de conocimiento.

## Flujo de Trabajo
1. **Entrada:** Los archivos a procesar se encuentran en la carpeta `data crudo`.
2. **Salida:** El archivo Markdown resultante DEBE guardarse en la carpeta data procesada.
3. Mueve el archivo de data crudo a carpeta 
Data Crudo procesada

## Instrucciones para el Agente

Cuando se te pida procesar un archivo de la carpeta `data_crudo`, tu objetivo es generar un resumen estructurado en formato Markdown.

**Pasos a seguir:**
1. Lee el archivo desde `data_crudo`.
2. Extrae e identifica la información requerida de uno o varios puntos
3. Crea documentos con información con siguientes puntos
    Que, quien, como, cuando, donde, porque, descripcion
Consulta si no tienes alguno de estos datos o añade en tabla faltante
4. segun .agents/skills/Matriz_de_clasificacion.md realiza la clasificacion de cada uno
5. añade parametro, "historia de usuario" y ponlo con valor "Para procesar" en cada uno
6. Genera el contenido Markdown con nombre "Requerimiento_" + nombre de cada uno y añade el un valor de Origen, y añade el valor Nombre de requerimiento con la información extraida y clasificada
7. Añade en la parte final la fuente de donde se extrajo la información
8. utiliza en el nuevo archivo .agents/skills/3.Escritura_obsidian.md

9. **Guarda el archivo resultante en la carpeta `data_procesada`** con un nombre descriptivo
